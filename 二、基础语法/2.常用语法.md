# 一、条件控制语句

`Python` 条件语句是通过一条或多条语句的执行结果(结果是 `true` 或 `false`) 来决定代码块的执行。

+ 语法

```python
if 表达式1：
    语句1
elif 表达式2：
    语句2
else:
    语句3
```

需要注意的是：

1. 每个条件后面需要使用 `:`，表示接下来是满足条件后要执行的语句；
2. `python` 使用缩进来划分语句块，相同缩进数的语句在一起组成一个语句块；
3. `python` 中没有 `switch ... case ...` 语句。

+ `if` 嵌套

```python
if 表达式1:
    语句1
    if 表达式1-1:
        语句1-1
    elif 表达式1-2:
        语句1-2
    else:
        语句
elif 表达式2:
    语句2
else:
    语句3
```

+ 代码

```python
import random
answer = random.randint(1,10)
temp = input('请输入一个数字：')

guess = int(temp)

if guess == answer:
    print('你猜对了')
    break
else:
    print('你猜错了')
print('游戏结束。。。')
```
